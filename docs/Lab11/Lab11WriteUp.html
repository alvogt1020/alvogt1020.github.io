1 Introduction

When performing point estimation, there isn’t just one right answer. For example, the Method of Moments (MOM) and Maximum Likelihood Estimates (MLEs) aren’t always the same.

One question that comes up, is why we calculate the sample variance as \[S_{n-1}^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i -\bar{X})^2,\] when it is supposed to be the ``average squared distance from the mean, i.e., \[S_n^2 = \frac{1}{n} \sum_{i=1}^n (X_i -\bar{X})^2.\]

Poisson:

Code 
set.seed(7272)

var.n1 = NA #initialized vector for S_{n-1}^2
var.n = NA #initialized vector for S_{n}^2
n = 20
var.true = 2

for(i in 1:1000){
  x = rpois(n = n, lambda = 2)
  mean.val = sum(x) / n
  sqr.dev = (x-mean.val)^2
  
  var.n1[i] = sum(sqr.dev) / (n-1)
  var.n[i] = sum(sqr.dev) / n
}

# Compare estimators
bias.n1 = mean(var.n1 - var.true)
bias.n = mean(var.n - var.true)

precision.n1 = 1/var(var.n1)
precision.n = 1/var(var.n)

mse.n1 = bias.n1^2 + var(var.n1)
mse.n = bias.n^2 + var(var.n)

# Create numerical table
poisson.comparison = tibble(
  Estimator = c("Variance (n-1)", "Variance (n)"),
  Bias = c(bias.n1, bias.n),
  Precision = c(precision.n1,precision.n),
  MSE = c(mse.n1, mse.n)
)

3 Executive Summary

Summarize the work you’ve done here to provide a brief (200-300 word) summary of the simulation and the results using your work in Lab 11. Your summary should be professional, clear, and all claims should be corroborated with statistical support.

Response: In the above lab, we compare two variance point estimators, \(S^2_{n-1}\) and \(S^2_{n}\) to identify why using the \(n-1\) version is standard practice. We compare the two point estimators through three primary measures (listed in order of priority): (1) Bias, (2) Precision, and (3) Mean Squared Error (MSE). That is to say, we first would like to compare each estimator’s bias: if both are unbiased, we compare the precision, if both are biased, we compare the MSE.

To obtain these values, we simulated 1,000 samples for four distributions: Poisson, Binomial, Normal, and Exponential. Each of these distributions is parameterized such that the true variance is 2, allowing for clear and consistent analysis. For each distribution, we also compared numerical and graphical summaries for multiple n values (20, 50, 100, 500).

The primary findings from this approach is that, regardless of distribution, \(S^2_{n}\) is consistently biased downward and \(S^2_{n-1}\) is consistently unbiased. This difference is more pronounced for small \(n\): as \(n\) increases for each of these distributions, the biases of the two estimators converge toward 0. Note that because only \(S^2_{n-1}\) is unbiased, we do not need to analyze MSE. Furthermore, precision increases for both estimators with \(n\), and the distributions for both estimators improve from right-skewed for small \(n\) to symmetric/approximately normal for large \(n\).

After obtaining our primary results, we study if outcomes change under a bootstrapping procedure, and if so, how. The predominant result from this analysis was that bootstrap sampling produces significantly more variable estimator distributions that change wildly with the sampling seed. The bootstrap results still loosely depicted the behavior we exhibited before, but with less reliability.

Overall, this exercise confirms the advantage of using \(S^2_{n-1}\) over \(S^2_{n}\), particularly when obtaining large samples is not feasible.
